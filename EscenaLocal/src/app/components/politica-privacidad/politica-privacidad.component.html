Política de Privacidad

Cómo gestionamos tus datos en EscenaLocal

Última actualización: 10/11/2025
Esta plataforma forma parte de un proyecto de tesis. Los datos se utilizan con fines académicos y de funcionamiento del sistema.
1. Introducción

Esta política explica qué datos recopilamos, para qué los usamos y cómo los protegemos. Al usar el sitio, el usuario acepta esta Política de Privacidad.

2. Datos que recopilamos
  • Datos de registro: usuario, contraseña, correo.
  • Datos de perfil de artista/productor: nombre artístico, redes, imagen.
  • Datos técnicos: IP, navegador, dispositivo.
  • Datos de eventos publicados.
3. Finalidad del tratamiento

Usamos los datos para permitir el registro, mostrar eventos, gestionar perfiles y mejorar la experiencia. No se venden datos a terceros.

4. Seguridad

Se aplican medidas razonables de seguridad para proteger la información almacenada. Sin embargo, ningún sistema es 100% seguro.

5. Cookies

El sitio puede usar cookies para recordar tu sesión o para estadísticas. Puedes desactivarlas desde tu navegador.

6. Derechos del usuario

Podés solicitar acceso, rectificación o eliminación de tus datos escribiendo a privacidad@escenalocal.com.

Si continúas usando el sitio, entendemos que aceptás esta política.
